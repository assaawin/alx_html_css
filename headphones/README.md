# Headphones - Web Page Implementation

This project is a responsive web page implementation for a headphones company, designed from scratch using pure HTML and CSS without any external frameworks or JavaScript.

## Project Overview

The webpage is a fully responsive implementation based on designs provided by Nicolas Philippot, UI/UX designer. It adapts to different screen sizes (mobile, tablet, and desktop) while maintaining visual consistency and accessibility standards.

## Design Specifications

### Colors
- Primary Color: #071629 (dark blue background)
- Accent Color: #FF6565 (red for hover states and buttons)
- Text Colors: White (#FFFFFF) and Dark Gray (#333333)

### Typography
- Primary Font: Source Sans Pro
- Icon Font: Spin-Cycle-OT (for special icons)
- Font Sizes: Responsive scaling based on screen size

### Layout
- Maximum content width: 1000px (centered)
- Mobile breakpoint: 480px
- Tablet breakpoint: 768px

### Interactive States
- **Links hover/active**: Color changes to #FF6565
- **Buttons hover/active**: Opacity reduces to 0.9
